# EPIC-02 Design System

## Mục tiêu

Xây dựng Design System thống nhất cho toàn bộ dự án Meliora.

Thiết lập các quy chuẩn về giao diện nhằm đảm bảo:

- Đồng nhất UI/UX.
- Dễ mở rộng.
- Dễ tái sử dụng.
- Dễ bảo trì.
- Tăng tốc độ phát triển Component.

Design System sẽ trở thành nền tảng cho tất cả Shared Components ở các Epic tiếp theo.

---

## Business Requirement

Meliora là hệ thống quản lý nên giao diện cần:

- Chuyên nghiệp.
- Hiện đại.
- Tối giản.
- Thân thiện với người dùng.
- Responsive trên nhiều thiết bị.
- Có khả năng mở rộng lâu dài.

---

## Deliverables

Sau khi hoàn thành Epic này, dự án sẽ có:

- Design Principles.
- Design Tokens.
- Color Palette.
- Typography System.
- Spacing System.
- Shadow System.
- Border Radius System.
- SCSS Architecture.
- Global Styles.
- Handbook Design System.

Đây sẽ là nền tảng để xây dựng toàn bộ Shared Components.
